Dialog

在保留当前页面状态的情况下，告知用户并承载相关操作。


基础用法

Dialog 弹出一个对话框，适合需要定制性更大的场景。

标题

这是一段内容

这是一段内容

这是一段内容


显示代码 

<div class="dialog_area">
    <button id="openDialog" class="btn primary">打开对话框</button>

    <div id="dialog" class="dialog hidden">
        <div class="dialog-content">
            <h2 class="dialog-title">标题</h1>
            <p class="dialog-text">这是一段内容</p>
            <p class="dialog-text">这是一段内容</p>
            <p class="dialog-text">这是一段内容</p>
            <div class="dialog-buttons">
                <button id="confirm" class="btn primary">确定</button>
                <button id="cancel" class="btn secondary">取消</button>
            </div>
        </div>
    </div>
</div>

<style>
.dialog {
    position: fixed;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
    background-color: rgba(0, 0, 0, 0.5);
    display: flex;
    justify-content: center;
    align-items: center;
}

.dialog-content {
    background-color: white;
    padding: 20px;
    width: 300px;
    display: flex;
    flex-direction: column;
    justify-content: space-between;
    border-radius: 10px;
    border: var(--color-border-medium) solid 1px;
}

.dialog-buttons {
    display: flex;
    justify-content: flex-end;
}

.dialog-buttons button {
    margin-left: 10px;
}

.hidden {
    display: none;
}
</style>

<script>
document.getElementById('openDialog').addEventListener('click', function () {
    document.getElementById('dialog').classList.remove('hidden');
});

document.getElementById('confirm').addEventListener('click', function () {
    document.getElementById('dialog').classList.add('hidden');
});

document.getElementById('cancel').addEventListener('click', function () {
    document.getElementById('dialog').classList.add('hidden');
});
</script>
