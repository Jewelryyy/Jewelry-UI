Layout 布局

Layout 布局组件，用于页面整体布局。

！！！该组件采用了 flex 布局，使用前请确定目标浏览器是否兼容。


常见页面布局

Header
Main
Footer

显示代码 

<header class="layout_header">Header</header>
<main class="layout_main">Main</main>
<footer class="layout_footer">Footer</footer>

<style>
.layout_header {
    /* 使内容居中 */
    display: flex;
    align-items: center;
    justify-content: center;
    background-color: #00717d;
    color: #F5F5F5;
    /* 设置页眉距离屏幕顶部的距离为0 */
    top: 0;
    /* 设置页眉的宽度为100%，使其占满整个屏幕宽度 */
    width: 100%;
    height: 10vh;
}

.layout_main {
    flex: 1;
    display: flex;
    align-items: center;
    justify-content: center;
    background-color: rgb(34, 141, 187);
    color: #F5F5F5;
    height: 20vh;
}

.layout_footer {
    display: flex;
    align-items: center;
    justify-content: center;
    background-color: #00717d;
    color: #F5F5F5;
    width: 100%;
    height: 10vh;
}
</style>
